# 1. 安装 htpasswd 工具 (如果未安装)

```
sudo apt update && sudo apt install apache2-utils 
```

# 2. 创建第一个用户 (例如 user_a)

如果是临时目录，就用/tmp ，其中，user_a就是用户名，让你输入的密码就是后续登录的时候需要使用的密码

```
sudo htpasswd -c /tmp/.htpasswd user_a
```

例如现在输入密码是：123456

```
New password: 123456
Re-type new password: 123456
Adding password for user user_a
```

那么可以得到这样的文件

```
cat /tmp/.htpasswd
user_a:$apr1$C7seoLe4$QgbJNdWMW5lxWiqXgKQz01
```

其中，用户名和密码就是 user_a:123456 , 而$apr1$C7seoLe4$QgbJNdWMW5lxWiqXgKQz01是经过哈希算法之后的值。

# 3. 添加更多用户 (不需要 -c 参数)

第一个用户需要 -c 参数。第二个用户不能用 -c ， 如果用了，那么第二个用户会覆盖第一个用户。

```
sudo htpasswd /tmp/.htpasswd user_b
```

假设我给user_b以同样的方法输入密码也是123456，相同的密码，获得的哈希居然不一样。

```
cat /tmp/.htpasswd
user_a:$apr1$C7seoLe4$QgbJNdWMW5lxWiqXgKQz01
user_b:$apr1$H5B4DFSE$92/wLe7qqZdTSPVjuq7ZU.
```
