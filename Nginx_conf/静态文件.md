网页挂载文件的版本 nginx.conf关键配置文件如下

```
server {
    listen      80;
    server_name localhost;
    charset utf-8;
    root /data;
    location / {
        autoindex on;
        autoindex_exact_size on;
        autoindex_localtime on;
    }
}
```

如果需要通过sjwayrhz/nginx:stable创建网页映射文件的dockerfile，如下用例：

```
FROM sjwayrhz/nginx:stable
ADD ${files} /data
CMD ["nginx", "-g", "daemon off;"]
```
